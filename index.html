The Amazing JavaScript Kim1!

Hi, thanks for stopping by to play with this virtual Kim-1. If you aren't at all familiar with this machine, I suggest checking out Wikipedia for an overview.

I must extend a HUGE debt of gratitude to Stian Soreng for both writing and GPLing the actual 6502 emulator that runs this. Additional thanks to Nick Morgan for the changes he made as well. I started out trying to write my own but was in over my head. Without this excellent code, this project would not have been possible.

This is actually my entry for the 2013 Retrochallenge, and as such was hacked together in a real hurry. Also, I knew nothing of 6502 assembly before starting, and I've never even held a Kim-1 before. This should be a big red flag that this thing is pretty limited. That said, I worked through some of the example programs in The First Book of Kim and many of the shorter programs work. The ROM should be completely loaded, but obviously things like hardware addresses will not work. This is best thought of as a terribly innefficent way to enter code into a generic CPU emulator, with a few hacks to make it behave like a Kim-1.

Quick enter a program. Each line should start with an address, and then space delimited values.
